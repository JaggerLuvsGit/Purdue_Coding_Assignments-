IEEE 802.3 (Ethernet)

IEEE 802.3

The Ethernet Standard

  • Purpose: Standard for wired LAN connections.
  • Key Features: Reliable, high-speed data transmission over cables.
  • Use Cases: Office networks, data centers.

    • IEEE 802.11 (Wi-Fi)

    IEEE 802.11

    The Wireless (WiFi) Standard

  • Purpose: Wireless LAN technology.
  • Key Features: Mobility, flexibility, scalability.
  • Use Cases: Home networks, public hotspots, enterprise environments.

    • IEEE 802.1Q (VLAN)

    IEEE 802.1Q

    The VLAN Standard

  • Purpose: Virtual LAN for network segmentation.
  • Key Features: Improved security, traffic management, resource optimization.
  • Use Cases: Large organizations, multi-tenant environments.

    • IEEE 802.15 (Bluetooth)

    IEEE 802.15

    The Bluetooth Standard

  • Purpose: Short-range wireless communication.
  • Key Features: Low power consumption, simplicity.
  • Use Cases: Wireless peripherals (e.g., keyboards, mice), IoT devices.

    • DNS (Domain Name System)

    Domain Name System

    The DNS Standard

  • Purpose: Translates domain names into IP addresses.
  • Key Features: Distributed architecture, hierarchical structure.
  • Use Cases: Web browsing, email delivery.

    • IPv4 vs. IPv6

    IPv4 vs. IPv6

    The IPV4 and IPV6 Standard

  • Purpose: Internet Protocol for identifying and locating devices on a network.
  • Key Features: IPv4 - 32-bit address space, IPv6 - 128-bit address space, improved security and efficiency.
  • Use Cases: Global internet addressing, IoT deployment.

    • HTTP vs. HTTPS

    HTTP vs. HTTPS

    The HTTP and HTTPS Standard

  • Purpose: Protocols for transferring data over the internet.
  • Key Features: HTTP - unsecured, HTTPS - secured (encrypted).
  • Use Cases: Website access, online transactions.

    • SSH (Secure Shell)

    Secure Shell

    The SSH Standard

  • Purpose: Secure remote access and control.
  • Key Features: Encrypted communication, authentication methods.
  • Use Cases: Remote server management, secure file transfer.

    • FTP (File Transfer Protocol)

    File Transfer Protocol

    The FTP Standard

  • Purpose: Protocol for transferring files between computers.
  • Key Features: Support for large file transfers, authentication.
  • Use Cases: Uploading website files, sharing documents.
